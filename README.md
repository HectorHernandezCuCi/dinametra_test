## 🚀 Setting up the Backend

1. **Required environment variables for the backend (.env):**

NASA_API=YOUR_API_KEY

Run FastAPI server
uvicorn app.main:app --host 0.0.0.0 --port $PORT

2. **Update frontend environment variables:**
NEXT_PUBLIC_API_URL=https://dinametra-test.onrender.com

Run NextJS server
npm run dev


## 🏁 Getting started

1. **Clone the repository:**

```bash
https://github.com/HectorHernandezCuCi/dinametra_test.git
```

Project structure:
```
dinametra_test/
├── dinametra/    # Frontend React application
└── backend/     # Backend API server
```

### Installation

Install dependencies:
```bash
npm install
```

### Development
Choose one of these methods to start the development server:

1. **Standard development:**
```bash
npm run dev
```

2. **FastAPI:**
```bash
uvicorn app.main:app --host 0.0.0.0 --port $PORT
```
